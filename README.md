# Aplicacion basica de peliculas
API - REST de peliculas del laboratorio de la universidad uniremington

# Instalacion de paquetes
composer install 

# Luego abren el administrador que esten trabajando (En nuestro caso es phpmyadmin)
Recuerden que deben abrir el phpmyadmin, si instalaron XAMPP pueden ingressar por: localhost/phpmyadmin

# Crear la base de datos
Crean la base de datos con el nombre: peliculapp

# Entran por la linea de comando y abren la carpeta del proyecto

# Ejecutan las migraciones
php artisan migrate
